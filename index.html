Maize Disease Detection

This project is a web application that leverages deep learning to identify diseases in maize leaves from uploaded images. Built with a Flask backend and a Keras/TensorFlow model, it allows farmers and agricultural experts to quickly diagnose plant health issues. The model was trained on a large dataset of maize leaf images and achieves high accuracy in classifying common diseases. The user interface is simple and mobile-friendly, making it accessible in the field.

Challenges & Solutions

  • Challenge: Insufficient labeled images for certain rare maize diseases.
    Solution: Used data augmentation and sourced additional open datasets to improve model generalization and accuracy.
  • Challenge: Ensuring the model runs efficiently on low-resource devices in the field.
    Solution: Optimized the model and reduced its size for faster inference and lower memory usage.

Smart Sentiment Analysis System (Ongoing)

This ongoing project for Fidelity Bank Ghana analyzes customer emotions in real time using facial recognition and AI. It uses a ResNet-50 model with TensorFlow and OpenCV to detect emotions such as happy, neutral, and angry, achieving 88% accuracy so far. The system integrates with the bank's CRM via REST API, enabling automated tagging of customer feedback and improving service response. User acceptance testing has shown positive results among staff.

Challenges & Solutions

  • Challenge: Limited dataset for training the emotion detection model led to initial low accuracy.
    Solution: Expanded the dataset by collecting more customer images and applied data augmentation techniques, which improved model performance.
  • Challenge: Integration issues between the AI system and the bank's CRM.
    Solution: Developed a custom REST API layer and worked closely with the CRM team to ensure seamless data exchange.
  • Challenge: Real-time processing speed was initially slow.
    Solution: Optimized the model and code for faster inference and leveraged hardware acceleration where available.
