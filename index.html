Maize Disease Detection

A web app that uses deep learning to identify diseases in maize leaves from uploaded photos. Built with Flask and a Keras/TensorFlow model, it gives farmers and agricultural experts a fast, mobile-friendly way to diagnose plant health in the field.

Challenges & solutions

  • Limited labeled data — Used data augmentation and sourced open datasets to improve model generalization and boost accuracy on rare disease classes.
  • Field performance — Optimized and compressed the model for faster inference and reduced memory usage on low-resource devices.
Smart Sentiment Analysis System

An ongoing system for Fidelity Bank Ghana that detects customer emotions in real time using facial recognition. ResNet-50 + TensorFlow + OpenCV classify emotions as happy, neutral, or angry — currently at 88% accuracy — and feed results into the bank's CRM via REST API.

Challenges & solutions

  • Low initial accuracy — Expanded the training dataset with augmentation, significantly improving model performance across all emotion classes.
  • CRM integration friction — Built a custom REST API layer and collaborated with the CRM team to ensure clean, reliable data exchange.
  • Real-time latency — Optimized the inference pipeline and enabled hardware acceleration, bringing response time to an acceptable level for live use.
Complaint Management System

A full-stack complaint tracking and resolution platform built with Django and MySQL for GCTU. Features role-based access, automated notifications, analytics dashboards, and a responsive UI designed in Figma and implemented with Bootstrap.

Challenges & solutions

  • Concurrent submissions — Implemented robust transaction management and database locking in Django to guarantee data consistency under load.
  • Non-technical users — Ran iterative user-testing sessions and refined the UI/UX using Figma prototypes, improving adoption by 35%.
